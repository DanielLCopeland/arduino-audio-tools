# ADS1015

![ADS1015](https://pschatzmann.github.io/arduino-sound-tools/doc/resources/ads1015.jpeg)

The ADS1015 is a 12 bit ADC which is connected via I2C.

### Pins:
 
| ADS1015 | ESP32
|---------|---------------
| VDD     | 3.3
| GND     | GND
| SCL     | 
| SDA     | 
| ADDR    | 
| ALRT    | 
| A0      | Instrument
